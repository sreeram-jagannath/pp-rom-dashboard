# Parts Pricing Romania

Streamlit dashboard

### Steps
- install libraries in requirements.txt
- place the data file (name it as `merged_data.csv`) in the same location as `app.py`
- type `run` in command prompt

### Folder structure
```
project
│   README.md
│   .gitignore   
│   requirements.txt
|   app.py
|   run.bat
└───data
│   │   merged_data.csv
│   │   model_results.xlsx
│   │   reconciled_data.csv
```
